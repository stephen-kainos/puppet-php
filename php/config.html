Overview

Configure php.ini settings for a PHP SAPI

Parameters

file

The path to ini file

config

Nested hash of key => value to apply to php.ini

Examples

php::config { '$unique-name':
  file  => '$full_path_to_ini_file'
  config => {
    {'Date/date.timezone' => 'Europe/Berlin'}
  }
}
